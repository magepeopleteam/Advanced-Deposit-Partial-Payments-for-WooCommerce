Advanced Partial/Deposit Payment For Woocommerce is a Deposit Plugin for WordPress which is based on WooCommerce. 

== Description ==
Advanced Partial/Deposit Payment For Woocommerce- is one of the best and simple deposit/partial payment plugin for Woocommerce. You can add partial deposit amount system in your Woocommerce shop, Easily and smartly. User can pay rest of the amount from my account page any time and in any payment gateway later. Its also support Woocommerce Event Manager plugin, so you can easily enable partial/deposit amount feature in your event. 


###  Free Version Plugin Features:

➡ Support All Payment Method available in WooCommerce.
➡ Cross Browser Support – (Chrome, Firefox, Safari, Opera, Etc.)
➡ Multilingual Translation Support
➡ Partial Payment History in My account page.
➡ Fixed/Parcent Type Deposit/partial payment option.
➡ Pay due amount from My Account Page & support all payment gateway.
➡ Woocommerce Event Manager Plugin Supported
➡ Payment History in My Account Page

###  How to use:

After successfully installed and activate you can see a new tab name Deposit in the Woocommerce Product Section. Open the tab then tick to enable the Deposit payment open then select Deposit type and enter the deposit value & update the product. After successfully placed a deposit order you can see the details in the order details page.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
